# BrainHack2020 - RBG Trees
**1. Create conda environment**
conda create -n BH2020_env python=3.7

**2. Activate environment**
activate BH2020_env

**3. Install dependencies**
pip install -r requirements.txt  
requirements are combined for both required task

**4. Task**
There's 2 main components:
  1. Image recognition (CV)  
  2. Natural Language Processing (NLP)  
  
Work within the folder for the component you are on

**Additional:**  
CV - Computer Vision  
NLP - Natural Language Processing
